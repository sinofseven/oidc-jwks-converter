# build_linux_armビルドエラーの修正

## 目的
ビルドできないと困る。

## 要望
build_linux_armでエラーが出た。修正して欲しい。

## 備考
CIのログは `ci.log` というなまえでタスクファイルと同じディレクトリに置いている。

## 完了サマリー

**完了日時**: 2026-05-18T16:47:00+09:00

### 修正内容

`.github/workflows/build.yml` の `build-linux-arm` ジョブから `--user 1001:1001` フラグを削除した。

**原因**: Docker コンテナ内の UID 1001 ユーザーが、マウントされたホスト側の `.cargo/registry/` ディレクトリへの書き込み権限を持たないため、Permission denied エラーが発生していた。

**解決方法**: Docker をroot ユーザーで実行することで、キャッシュディレクトリへの書き込みを可能にした。
