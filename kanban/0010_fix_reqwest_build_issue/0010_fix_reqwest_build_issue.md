# reqwestビルド問題の修正

## 目的
ビルドできなくて困っているため

## 要望
reqwestのバージョンを上げたらビルドが通らなくなった修正して欲しい。CIによるビルドログはタスクファイルと同じディレクトリの `ci_log` に置いている。

## プラン
reqwest 0.13 で `rustls-tls` フィーチャーが廃止され `rustls` に変更されたため、Cargo.toml を修正

## 完了サマリー
2026-05-18T18:30:00+09:00

修正内容：Cargo.toml の reqwest フィーチャー設定を `["json", "rustls-tls"]` から `["json", "rustls"]` に変更

結果：
- ビルド成功 ✓
- ユニットテスト 14 / 14 成功 ✓
- 統合テスト 3 / 3 成功 ✓
