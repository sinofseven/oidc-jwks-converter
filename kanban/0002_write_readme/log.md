### main.rs の機能
- **CLI引数**: JWKS URL（必須）、--output (オプション)
- **処理フロー**:
  1. JWKS URL から JWKS データを取得
  2. 各キーを PEM フォーマットに変換
  3. `{key_id}.pem` というファイル名で出力
- **エラーハンドリング**: サポートされていないキータイプは警告で続行

### converter.rs の機能
- **変換対象**: RSA、EC（楕円曲線）の2種類
- **エラー型**: ConversionError で詳細なエラー情報を提供
- **RSA処理**: `n` (modulus) と `e` (exponent) から RsaPublicKey を構築
- **EC処理**: `x`, `y` 座標から P256PublicKey を構築、SEC1フォーマット対応

### jwks.rs の機能
- **JWKS データ構造**:
  - JwksResponse: keys フィールドを持つ
  - Jwk: 各キーの情報（kty, kid, use, alg, n, e, crv, x, y）
- **取得方法**: reqwest で非同期HTTP GET

### キータイプと署名アルゴリズム
- **RSA**: RS256, RS384, RS512（署名アルゴリズムは alg フィールド）
- **EC**: ES256, ES384, ES512（曲線は crv フィールド）
